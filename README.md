# 📰 News Portal System

A dynamic and responsive News Portal web application built using **PHP** and **MySQL**, designed to manage and display news articles efficiently with an admin dashboard.

---

## 🚀 Features

- 🗂️ Category-based news management  
- 🛠️ Admin dashboard for CRUD operations (Create, Read, Update, Delete)  
- 📱 Responsive user interface for all devices  
- 🔍 Organized content display for better readability  
- 🧾 Structured database for efficient data handling  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL
